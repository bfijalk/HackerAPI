# HackerAPI

## How to run

### Requirements
- DotNet Core SDK 8.0

1. Copy repository content and switch to Main branch
2. Open your favourite console tool (e.g. cmd, terminal, bash) and got to directory where you downloaded repository content
3. Go into the ~/Hacker.API/Hacker.API folder
4. execute command `dotnet run`
5. Your API should be ready to use now. You can test it by making a GET request to the following endpoint
    `http://localhost:5108/HackerNews?count=<n>` e.g. `http://localhost:5108/HackerNews?count=3`

    You can use swagger UI as well: http://localhost:5108/swagger
